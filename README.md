# 플러터 개발자 커뮤니티앱 

- AWS 서버 연동 시험용 프로젝트
- 커뮤니티
- 하이브리드앱(Ios , Android 동시개발가능)


### 작업 소요 기간 

- 서버+클라이언트 = 30일
- 클라이언트(Flutter) 코드 : https://github.com/LandWhale2/Flutter-Developer_community

### 사용기술 

- Flutter
- Dart
- AWS
- Docker
- Django
